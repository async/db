serve

Separate dev-tool routes from app-facing data routes.

The built-in server binds to loopback by default. It is local development infrastructure for REST, GraphQL, viewer metadata, CSV import, schema inspection, and live diagnostics.

Route families

RouteDefaultUse
/__dbtool route baseViewer, schema, manifest, batch, import, log, and events.
/db/users.jsonapp data aliasFixture-like REST reads from synced runtime state.
/graphqlGraphQL endpointDependency-free GraphQL subset for local data reads and mutations.
/__db/operations/:refoperation executionAllowlisted REST or GraphQL templates when operations are enabled.

Move route bases deliberately

export default defineConfig({
  server: {
    apiBase: '/__db',
    dataPath: '/db',
    host: '127.0.0.1',
    port: 7331,
  },
});

Harden exposure by route family

export default defineConfig({
  server: {
    expose: {
      rest: 'registered-only',
      graphql: false,
      viewer: 'dev',
      schema: 'dev',
      manifest: 'dev',
    },
  },
});
